Install Kattour.

Start a Kattour project, run the dev server, build static output, and prepare production deployment.

Global install

npm install -g kattour

Create app

kattour create my-app
cd my-app
kattour dev app.kat

Build

kattour build app.kat

Deploy

dist/index.html
