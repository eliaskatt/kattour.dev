Create Stunning Interfaces with Kattour

Kattour is a declarative, minimal, YAML-inspired programming language for designing modern user interfaces quickly and clearly.

🌟 Why Kattour?

  • ⚡ Blazing fast prototyping for websites and apps
  • 📦 Lightweight syntax focused only on UI logic
  • 💡 Easy to learn: no HTML, no CSS, no JS needed
  • 📄 Readable and clean like writing in plain English
  • 🔁 Built-in support for if, for, and data

🧠 What does it look like?

data:
  user: "Elias"

if user:
  text:
    content: "Welcome, $user"

button:
  label: "Logout"

🚀 Build with 3 commands

kattour init
kattour build
kattour serve
