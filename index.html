Build UIs with clarity.

Kattour is a declarative UI language designed to make interfaces readable first — with state, computed values, effects, routing, hydration, and compiler-first development built into the platform.

Install Kattour → View Examples 
page Home

state count = 0
computed label = "Count: " + $count

route "/" {
  screen {
    column {
      title "Hello Kattour"
      button "$label" {
        click count++
      }
    }
  }
}

Start fast.

The npm package foundation is being prepared for global installs. The current public site is an alpha preview.

npm install -g kattour
kattour create my-app
cd my-app
kattour dev app.kat

Syntax that stays obvious.

Kattour avoids YAML dependency and JSX noise. The language is designed around direct human intent.

State

state username = "Elias"

Binding

input { bind username }

Routes

route "/user/:name" { title "$route.params.name" }
