Clarity-first UI language.

Kattour is a human-readable declarative UI language for building web, mobile, and desktop interfaces with less boilerplate, cleaner state, routing, effects, and a fast compiler-first workflow.

page Home

state count = 0
computed label = "Count: " + $count

route "/" {
  screen {
    column {
      title "Hello Kattour"
      button "$label" {
        click count++
      }
    }
  }
}

Readable by default

No JSX noise. No YAML dependency. Kattour syntax is built around human intent.

Reactive runtime

State, computed values, effects, bindings, routing, hydration, and route params are built in.

Compiler-first

Compile Kattour into clean HTML with a runtime today, then target more platforms later.

Start fast.

npm install -g kattour
kattour create my-app
cd my-app
kattour dev app.kat

Syntax that stays obvious.

State

state username = "Elias"

Binding

input { bind username }

Routes

route "/user/:name" { title "$route.params.name" }
