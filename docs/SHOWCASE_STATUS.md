# Showcase Status

## Already visible in the repository
- a new `apps/website` path for the future website app
- a new `apps/playground` path for the future playground app
- compiler, runtime, cli, stdlib, and ui package directories
- initial Kattour examples
- language, architecture, and roadmap documents

## Already implemented at a foundational level
- tokenizer
- parser
- diagnostics formatter
- HTML preview renderer
- browser playground shell
- website homepage shell

## Not fully wired yet
- package dependencies
- final build scripts
- public compiler entry cleanup
- migration from legacy root static pages to the new app path

## Meaning
Kattour is no longer only a static showcase. It now has the beginnings of a real product structure.
