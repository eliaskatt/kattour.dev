Documentation

Learn Kattour from zero to production UI. The docs are being built as interactive pages with live examples.

Getting Started

npm install -g kattour
kattour create my-app
cd my-app
kattour dev app.kat

Syntax

Kattour is not YAML. It uses a clean block-based declarative syntax focused on UI intent.

page Home

route "/" {
  column {
    title "Hello"
    button "Start"
  }
}

State

state count = 0
computed label = "Count: " + $count

button "$label" {
  click count++
}

Routing

route "/user/:name" {
  title "$route.params.name"
}

Effects

effect username {
  log "$username changed"
}
