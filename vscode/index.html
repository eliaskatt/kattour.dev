Install Kattour in VSCode.

The Kattour extension adds .kat and .kattour file support, syntax highlighting, snippets, diagnostics, bracket pairing, formatting and a command to open the live playground.

Step 1

Build the extension

Run these commands from the repository root to build and package the extension locally.

cd tools/vscode-extension
npm install
npm run build
npm run package
Step 2

Install the VSIX

The package command creates a .vsix file. Install it from the VSCode Extensions screen.

VSCode → Extensions
... menu
Install from VSIX...
Select kattour-language-0.0.1.vsix
Step 3

Create a .kat file

Open any .kat or .kattour file and the language features activate automatically.

page Home

route "/" {
  column {
    title "Hello Kattour"
    button "Start"
  }
}

Extension features

The current extension foundation is intentionally strict: it supports the new Kattour block syntax and rejects old YAML-like patterns.

.kat supportNative language mode for Kattour files.
Syntax highlightingKeywords, components, events, strings and variables.
SnippetsPage, route, component, form and stats starters.
FormatterBasic indentation formatter for block syntax.
DiagnosticsDetects legacy syntax, brace issues and missing page declaration.
CommandsFormat file and open playground.
BracketsAuto-closing pairs and folding markers.
FutureAutocomplete, hover docs, compile current file and quick fixes.

Open the Command Palette and run this command while editing a Kattour file.
