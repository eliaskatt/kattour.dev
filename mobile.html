Use Capacitor to convert your static Kattour app into a native mobile app.

1. Install Capacitor

npm install --global @capacitor/cli

2. Initialize Project

npx cap init
# App name: kattour
# Package ID: com.kattour.app

3. Add Platforms

npx cap add android
npx cap add ios

Now open Android Studio or Xcode to run and build your mobile app.
