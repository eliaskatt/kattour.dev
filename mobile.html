Use Capacitor to convert your static Kattour app into a native mobile app.

1. Install Capacitor CLI

Add Capacitor's command-line tool to your system.

npm install --global @capacitor/cli

2. Initialize Capacitor in Your Project

Run the init command inside your Kattour project's root folder.

npx cap init
# App name: MyKattourApp
# Package ID: com.kattour.app

3. Add Mobile Platforms

Add the native Android and iOS platforms to your project.

npx cap add android
npx cap add ios

Now open your project in Android Studio or Xcode to build and run your native mobile app!
