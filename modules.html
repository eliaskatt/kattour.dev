13. Accordion / FAQ

module: Disclosure
type: 'accordion'
items:
  - question: 'What is Kattour UI?'
    answer: 'It is a UI component library...'
  - question: 'Is it free to use?'
    answer: 'Yes, it is completely free...'
  - question: 'How do I install it?'
    answer: 'Just copy and paste the code...'

Kattour UI is a collection of professionally designed, fully responsive, and accessible UI components that you can drop into your projects. It's built with Tailwind CSS and Alpine.js for maximum flexibility and ease of use.

Simply find the component you need, click the "Copy" button to copy the HTML, and paste it into your project. Make sure you have Tailwind CSS included in your project for the styles to apply correctly.
