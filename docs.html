📘 Getting Started

Learn how to install and use Kattour in a few simple steps.

🚀 Step 1: Install the CLI

Install the global command-line tool via npm.

npm install -g kattour

🛠 Step 2: Create and Run a Project

Initialize, build, and serve your new project with three commands.

kattour init
kattour build
kattour serve

🔤 Step 3: Syntax Basics

Define UI elements like text, inputs, and buttons with simple key-value pairs.

text:
  content: "Hello, World!"

input:
  name: "email"

button:
  label: "Submit"

🔁 Step 4: Logic and Data

Use data and conditional logic to create dynamic interfaces.

data:
  loggedIn: true

if loggedIn:
  text:
    content: "Welcome back!"
else:
  text:
    content: "Please log in"
