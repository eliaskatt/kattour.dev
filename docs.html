📘 Kattour Language Documentation

The official guide for using the Kattour programming language.

🚀 Getting Started

Install the Kattour CLI globally using npm:

npm install -g kattour

Create and run a Kattour project:

kattour init
kattour build
kattour serve

📄 Syntax Overview

Kattour uses a clean, indentation-based structure similar to YAML.

Text Element

text:
  content: "Welcome to Kattour!"

Button Element

button:
  label: "Click Me"

Input Element

input:
  name: "username"

Image Element

image:
  src: "assets/logo.png"

🔁 Looping

Use for to repeat an element multiple times:

for 3:
  button:
    label: "Repeat"

🔂 Conditional Logic

Use data to define boolean variables and control logic with if and else.

data:
  isAdmin: true

if isAdmin:
  text:
    content: "Admin Panel"

else:
  text:
    content: "Access Denied"

📦 Project Structure

Typical Kattour project layout:

main.kt       # Main Kattour source file
build/        # Output directory
assets/       # Static files like images or stylesheets
