📘 Getting Started with Kattour

🚀 Install CLI

npm install -g kattour

🛠 Create a Project

kattour init
kattour build
kattour serve

🔤 Syntax Basics

text:
  content: "Hello"

input:
  name: "email"

button:
  label: "Submit"

🔁 Logic and Data

data:
  loggedIn: true

if loggedIn:
  text:
    content: "Welcome!"

else:
  text:
    content: "Please log in"
